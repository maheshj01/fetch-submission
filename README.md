### We Love Dogs

This is a simple React application that allows users to view dogs and match them.

Deployed on Vercel: https://fetch-loves-dogs.vercel.app/

### Technologies Used

- React
- Redux
- TypeScript
- Tailwind CSS
- Shadcn UI
- React Router
- Fetch API

### How to run the project

1. Clone the repository
2. Run `npm install`
3. Run `npm run start`
